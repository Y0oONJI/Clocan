## 설문지 개요

이 설문지는 Clocan 서비스의 핵심 사용자 시나리오 5가지에 대한 사용자 피드백을 수집하기 위해 작성되었습니다. 각 시나리오를 경험하신 후 아래 질문에 답변해 주시기 바랍니다.

### 시나리오 목록

1. **시나리오 1: 데모 체험 (비회원)** - 랜딩 → 퀴즈 → 결과
2. **시나리오 2: API 테스트** - 랜딩 페이지에서 API 테스트
3. **시나리오 3: 어드민 대시보드 접근** - 랜딩 → 어드민 대시보드
4. **시나리오 4: 회원가입 후 데모 체험** - 랜딩 → 회원가입 → 퀴즈 → 결과 (향후 구현)
5. **시나리오 5: 로그인 후 데모 체험** - 랜딩 → 로그인 → 퀴즈 → 결과 (향후 구현)
