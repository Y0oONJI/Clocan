## 1. 첫 방문 사용자 온보딩 플로우

### 🎬 시나리오: 신규 사용자의 첫 방문

**목표:** 사용자가 서비스를 이해하고 스타일 퀴즈를 시작하도록 유도

```
┌─────────────────────────────────────────────────────────────┐
│ STEP 1: 랜딩 페이지 진입 (/)                                │
├─────────────────────────────────────────────────────────────┤
│ • 화면: 환영 메시지 + 브랜드 아이덴티티                      │
│ • 구성요소:                                                  │
│   - Hero Section: "Closet Canvas" 타이틀                    │
│   - 가치 제안: 3개의 Feature Cards                          │
│     ✓ Find Your Style (스타일 발견)                         │
│     ✓ Curate Your Closet (옷장 큐레이션)                   │
│     ✓ Get Inspired (AI 추천)                               │
│ • 주요 액션: "Start Your Style Journey" CTA 버튼            │
│                                                              │
│ 예상 체류 시간: 10-30초                                      │
│ 이탈률 목표: < 40%                                          │
└─────────────────────────────────────────────────────────────┘
                             ↓ 클릭
┌─────────────────────────────────────────────────────────────┐
│ STEP 2: 스타일 퀴즈 시작 (/style-quiz)                      │
├─────────────────────────────────────────────────────────────┤
│ • 화면: 소개 화면                                            │
│ • 메시지: "Find Your Personal Style"                        │
│ • 설명: "Complete this quick quiz..."                       │
│ • Progress Bar: 0%                                           │
│ • 액션: "Let's start!" 버튼                                 │
│                                                              │
│ 예상 체류 시간: 5-10초                                       │
│ 전환율 목표: > 80%                                          │
└─────────────────────────────────────────────────────────────┘
```

**핵심 UX 고려사항:**
- ✅ **명확한 가치 전달**: 3가지 핵심 기능을 카드로 시각화
- ✅ **단일 CTA**: 사용자 선택의 혼란 최소화
- ✅ **시각적 계층**: 타이틀 → 설명 → 기능 → 액션 순서
- ✅ **반응형 디자인**: 모든 디바이스에서 동일한 경험

## 2. 스타일 퀴즈 완료 플로우

### 🎨 시나리오: 스타일 선호도 수집

**목표:** 사용자의 스타일 선호도를 재미있고 직관적으로 수집

```
┌─────────────────────────────────────────────────────────────┐
│ STEP 3: 스타일 선택 (Step 1/3)                              │
├─────────────────────────────────────────────────────────────┤
│ • 질문: "Which styles are you drawn to?"                    │
│ • 선택지: 6개 스타일 카드 (이미지 + 라벨)                    │
│   - Modern, Vintage, Bohemian, Streetwear, Classic,        │
│     Minimalist                                              │
│ • 인터랙션:                                                  │
│   - 클릭 → 선택 (링 강조 + 체크마크)                        │
│   - 다중 선택 가능                                          │
│ • Progress Bar: 33%                                          │
│ • 유효성 검사: 최소 1개 이상 선택 필수                       │
│ • 액션: "Next" 버튼 (선택 전 비활성화)                      │
│                                                              │
│ 예상 체류 시간: 20-40초                                      │
│ 평균 선택 개수: 2-3개                                        │
└─────────────────────────────────────────────────────────────┘
                             ↓
┌─────────────────────────────────────────────────────────────┐
│ STEP 4: 색상 선택 (Step 2/3)                                │
├─────────────────────────────────────────────────────────────┤
│ • 질문: "What colors do you prefer?"                        │
│ • 선택지: 6개 컬러 팔레트 카드                               │
│   - Neutrals, Pastels, Brights, Monochrome, Earthy,        │
│     Jewel Tones                                             │
│ • 인터랙션: 동일 (다중 선택)                                 │
│ • Progress Bar: 66%                                          │
│ • Back 버튼: 이전 단계로 이동 가능                          │
│                                                              │
│ 예상 체류 시간: 15-30초                                      │
└─────────────────────────────────────────────────────────────┘
                             ↓
┌─────────────────────────────────────────────────────────────┐
│ STEP 5: 영감 이미지 선택 (Step 3/3)                         │
├─────────────────────────────────────────────────────────────┤
│ • 질문: "Get Inspired - Select outfits you would wear"     │
│ • 선택지: 9개 outfit 이미지 (그리드 레이아웃)               │
│ • 인터랙션: 이미지 클릭 → 오버레이 + 체크마크               │
│ • Progress Bar: 100%                                         │
│                                                              │
│ 예상 체류 시간: 30-60초                                      │
│ 평균 선택 개수: 3-5개                                        │
└─────────────────────────────────────────────────────────────┘
                             ↓
┌─────────────────────────────────────────────────────────────┐
│ STEP 6: 완료 화면                                            │
├─────────────────────────────────────────────────────────────┤
│ • 화면: 축하 메시지                                          │
│ • 아이콘: 🎉 PartyPopper                                    │
│ • 메시지: "You're All Set!"                                 │
│ • 설명: "We've personalized your experience..."            │
│ • 액션: "Explore Your Closet Canvas" 버튼                   │
│                                                              │
│ 예상 체류 시간: 3-5초                                        │
│ 전환율 목표: > 95%                                          │
└─────────────────────────────────────────────────────────────┘
```

**핵심 UX 고려사항:**
- ✅ **Progress Indicator**: 항상 현재 진행 상황 표시
- ✅ **즉각적 피드백**: 선택 시 시각적 변화 (링, 체크마크)
- ✅ **자유로운 탐색**: Back 버튼으로 이전 단계 수정 가능
- ✅ **부담 없는 선택**: "1개 이상" 선택으로 낮은 진입 장벽
- ✅ **시각적 즐거움**: 이미지 중심 인터페이스

## 3. 결과 확인 및 재시도 플로우

### 📊 시나리오: 스타일 분석 결과 확인

**목표:** AI 분석 결과를 보여주고 다음 단계로 유도

```
┌─────────────────────────────────────────────────────────────┐
│ STEP 7: 결과 페이지 진입 (/style-quiz/result)               │
├─────────────────────────────────────────────────────────────┤
│ • URL 파라미터로 선택 데이터 전달                            │
│   ?styles=modern,vintage&colors=neutrals&inspirations=...   │
│                                                              │
│ • 로딩 상태 (2초):                                           │
│   - 스피너 애니메이션                                        │
│   - "Analyzing your style preferences..." 메시지            │
│                                                              │
│ • 심리적 효과: 개인화된 분석 진행 중이라는 인식             │
└─────────────────────────────────────────────────────────────┘
                             ↓
┌─────────────────────────────────────────────────────────────┐
│ STEP 8: 결과 표시                                            │
├─────────────────────────────────────────────────────────────┤
│ • 레이아웃 (상단 → 하단):                                    │
│                                                              │
│   1. Header                                                  │
│      - 아이콘: ✨ Sparkles                                  │
│      - 타이틀: "Your Style Profile"                         │
│                                                              │
│   2. Selection Summary (3개 카드)                           │
│      - Style Preferences: Badge로 선택 항목 표시            │
│      - Color Palettes: Badge로 선택 항목 표시               │
│      - Inspirations: 선택 개수 표시                         │
│                                                              │
│   3. AI Style Analysis (하이라이트 카드)                    │
│      - 개인화된 스타일 분석 텍스트                          │
│      - 3-4개 문단으로 구성된 상세 분석                      │
│                                                              │
│   4. Action Buttons                                          │
│      - 🔄 "Retake Quiz": 퀴즈 다시 하기                    │
│      - 🏠 "Back to Home": 홈으로 돌아가기                  │
│                                                              │
│ 예상 체류 시간: 40-90초                                      │
│ 재시도율 목표: 15-25%                                       │
└─────────────────────────────────────────────────────────────┘
```

**핵심 UX 고려사항:**
- ✅ **기대감 조성**: 로딩 애니메이션으로 가치 인식 증대
- ✅ **정보 계층**: 요약 → 상세 분석 순서로 스캔 가능성 향상
- ✅ **명확한 선택지**: 재시도 vs 홈 복귀, 두 가지 명확한 경로
- ✅ **공유 가능성**: URL에 결과 포함으로 북마크/공유 가능

## 5. 주요 UX 원칙

### 🎨 디자인 철학

1. **Progressive Disclosure (점진적 정보 공개)**
   - 한 번에 하나의 질문만 표시
   - 단계별로 정보 수집
   - 사용자의 인지 부하 최소화

2. **Immediate Feedback (즉각적 피드백)**
   - 선택 시 즉시 시각적 변화
   - Progress bar로 진행 상황 표시
   - 버튼 활성화/비활성화로 상태 명확화

3. **Forgiveness (용서하는 디자인)**
   - Back 버튼으로 이전 단계 수정 가능
   - "Retake Quiz"로 처음부터 다시 시작 가능
   - 선택 취소 자유로움

4. **Visual Hierarchy (시각적 계층)**
   - 큰 타이틀 → 설명 → 선택지 → 액션 버튼
   - 일관된 레이아웃 패턴
   - 명확한 CTA 강조

5. **Mobile-First Responsive (모바일 우선 반응형)**
   - 모든 화면 크기에서 최적화
   - 터치 친화적 인터랙션 영역
   - 그리드 레이아웃 자동 조정

## 📊 성공 지표 (Success Metrics)

### 핵심 KPI

| 지표 | 목표 | 측정 방법 |
|------|------|-----------|
| **퀴즈 완료율** | > 60% | (완료 수 / 시작 수) × 100 |
| **평균 완료 시간** | 2-4분 | 시작 ~ 완료 시간 측정 |
| **재시도율** | 15-25% | 결과 페이지에서 재시도 클릭 |
| **단계별 이탈률** | < 15% | 각 단계에서 이탈한 비율 |
| **모바일 사용률** | > 50% | 모바일 기기 접속 비율 |
